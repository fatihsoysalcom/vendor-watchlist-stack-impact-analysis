# Vendor Watchlist Stack Impact Analysis

This example simulates a 'vendor watchlist' analysis, demonstrating how potential future technology signals from various vendors might impact an existing software stack. It defines a current stack, lists hypothetical signals, and evaluates their relevance and potential changes required for the stack.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
